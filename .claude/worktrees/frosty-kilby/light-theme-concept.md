# Light Theme Concept: Ptolemaios Tactical UI

**Inspiration:** 
Mobile Suit Gundam 00 - Ptolemaios Bridge Screens

**Core Aesthetic:**
- Fictional sci-fi user interface (UI) design.
- Minimalist aesthetic, high-tech tactical display.
- 2D flat design, functional and industrial look.

**Color Scheme:**
- **Background:** Solid white (#FFFFFF).
- **Foreground:** Crisp black (#000000) lines and text.
- **Visuals:** Sharp typography, clean vector style.
- **Strict Rules:** No glows, no gradients.

**Key Elements:**
- Technical schematics of mecha/assets.
- Orbital trajectory maps (Graph links).
- Data charts and high-density information layout.
- Thin geometric HUD elements and UI framing.
