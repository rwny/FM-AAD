## 3. Key Features Reference
- **Refrigerant:** R32
- **Compressor:** Twin Rotary DC-Inverter
- **Coating:** Aqua Resin (Magic Coil)
- **Functions:** Self-Cleaning, Inverter Box (Sealed), Smart Wi-Fi (Optional)
